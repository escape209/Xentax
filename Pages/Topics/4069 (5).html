Post by revelation » Sun May 09, 2010 8:07 pm

The 360 file is named differently than the ps3 files, as such does not match with the way the original script file is attempting to match the filelist with the data bin files. rename them to work with the script as is, or modify the script to look for the names correctly.

ie. rename filelistu.x360.bin to filelist_imgu.x360.bin

Hope that helps.
