quickbms script

Code: Select all


get name BASENAME
string name + "_ext."
get name1 extension
string name + name1
get size long
get zsize asize
math zsize -= 4
savepos offset
comtype lz77wii
clog name 0 zsize zsize

seems like the creators of the wii game used modding tools for it?
