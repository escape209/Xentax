Post by Acewell » Tue Mar 19, 2019 8:08 am

masagrator wrote: ↑Fri Jan 11, 2019 2:13 pm Is it possible for someone to edit this script to work with uncompressed cases.big file?
big.nx is switch file...
this is relevant part from aluigi's BMS script with just big-endian removed to make it work with your Switch sample.

Code: Select all

# script for QuickBMS http://aluigi.altervista.org/quickbms.htm

goto -4
savepos OFFSET
get SIZE long
math OFFSET - SIZE
goto OFFSET
get ZERO long
get DUMMY long  # 3
get FILES long
for i = 0 < FILES
    get NAME_CRC long
    get OFFSET long
    get SIZE long
    get ZERO long
    get XSIZE long
    if XSIZE != 0
        math SIZE = XSIZE
    endif
    math OFFSET * 0x10
    log "" OFFSET SIZE
next i
