Saints Row The Third Remastered game was recently released. But I see nobody doing anything about this subject. Or nobody cares.

The extension of the files is in *.vpp_pc format. Previous rip tools don't work. *.vpp_pc format files should probably contain stuff like *.asm_pc or *.str2_pc.
Does anyone have any knowledge and ideas on this subject?
