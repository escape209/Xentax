Post by dj082502 » Mon Mar 05, 2012 2:33 am

Here's a NTP3 unpack quickbms script

Code: Select all

idstring NTP3
endian big
get version short
get files short
goto 0x10
for i = 0 < files
savepos offset
get size long
get name basename
string name + i
string name + .dds
log name offset size
math offset + size
goto offset
next i
I have altered chrrox's NTXR quickbms script.
