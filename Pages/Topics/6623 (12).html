Post by flatz » Thu Nov 24, 2011 9:47 pm

Here are steps needed to unpack cases.big.pc:
1) Install Python 2.7 from http://python.org/ftp/python/2.7.2/python-2.7.2.msi to C:\Python27, for example.
2) Download the my script and place it to: C:\tmp
3) Launch the command line and change the working directory to: C:\tmp
4) Then launch: python unpack_big.py C:\Games\L.A.Noire\final\pc\cases.big.pc > result.txt
After that you will get uncompressed files from cases.big.pc in the 'entries' folder. Besides, you can get unstructured files from some archives, for example, reftextures_populated.big.pc.
