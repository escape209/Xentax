Post by Evin » Sat Aug 25, 2012 9:28 am

I'm working on Heavy Rain Tools, to I can translate the game.

I'm very at the beginning of the develop/research! This is so far done:
- dat/d00/d01... unpacker
- segs unpacker/repacker

Because the tools are too big to upload here, available from here: https://dl.dropbox.com/u/4953836/Xentax ... inTools.7z
Because I don't have the header of the dat files with the size data of the inner files, I had to use search. The result of this search is the .fff files. The researching a very long progress, that's why I made these header files. Faster to use those to unpack files.

Future plan:
- texture unpack/repack (probably not all, I just want to find the fonts)
- text unpack/repack (so far I found 2 different format text files)
- dat repack (with size limit, because I can't edit the header of the dat files)


I found an interesting text format inside the game. Now I try to crack it, because until I can't edit that, there is no point to working on other parts of the game.
