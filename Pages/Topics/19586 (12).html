I believe that file list is generated whenever you run the program without any paramteres. To quote daemon1:
daemon1 wrote: ↑Sun Mar 03, 2019 1:02 pm 1. To make a list, run the tool without parameters
Example: Detroit > list.txt
It will create a list of potentially extractable game packages and their sizes (in bytes). MOST of them will have models, but there will be some with only videos, sounds or animations, these will be skipped with error message "not supported". (list.txt, you need this)

A list of all noticable models is included (Detroit_File_List.txt)
Btw that render was made in Cinema4D with Octane.
