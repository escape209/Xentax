The files are encoded with a simple algorithm. Here is the code of the decoder function (in Delphi):

Code: Select all

Procedure SpinTires_Decoder(pBuffer : Pointer;BufferSize : Integer);
Var I : Integer;
    p : ^Byte;
    B : Byte;
Begin
  p := pBuffer;
  For I := 0 To BufferSize - 1 Do
    Begin
      B := 251 - (I * 14);
      p^ := p^ + B;
      Inc(p);
    End;
End;

Post by chrrox » Thu Jun 06, 2013 11:42 pm

quickbms script to convert these files

Code: Select all

get name filename
string name + .xml
get size asize
set MEMORY_FILE binary ""
for i = 0 < size
set b i
math b * 14
set c 251
math c - b
get tmp byte
math tmp + c
math tmp & 0xFF
putVarChr MEMORY_FILE i tmp byte
next i
log name 0 size MEMORY_FILE
