Post by GRiNDERKILLER » Sun Nov 04, 2018 8:37 pm

Format is quite easy. I already have all weapons/modules converted to *.obj with my 010 HEX Editor Template. But... it has some fails like missing vertex normals because i don't know how to read/print them.

Here is sruct for static meshes without bones.

Code: Select all

int vertID;
int VBSize;
int VertexCount;

struct VTXBLOCK {

for (i=0;i<VertexCount;i++)

struct VTX {
        float VX;
        float VY;
        float VZ;
struct VTN {
        hfloat UNK0;
        float UNK1;
        float UNK2;
struct UV {
        hfloat VTU;
        hfloat VTV;
            }unitvector;
        }vertexnormal;
    }vertices;
}vtxblock;
