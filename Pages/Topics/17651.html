this bms script will decompress those bin samples

Code: Select all

# script for QuickBMS http://aluigi.altervista.org/quickbms.htm

comtype gzip
endian big
get ZSIZE long
get NAME basename
string NAME + _decomp.bin
clog NAME 0x4 ZSIZE ZSIZE
