Hi, there are some Driv3r/Driver-Threads around here, but no answers/reactions to these requests, so I started a new one. This game isn't that popular as the GTA series, but there are a lot of fans out there who want to mod the game, especially Driver Parallel Lines/Driver 4. There are some good websites which found out many things about the game's structures (me too^^). There are some tools for extracting/inserting textures (SLConverter by SparedLife) or converting the music files. But most fans want to mod the cars/add new 3D-models to the game. Unfortunately there are no skilled programmers in the community who could make a tool for extracting the game's archives. So here I am with my request.

I added parts of the archive "nyc_vehicles.sp" which contains models and textures of all the cars. There are some other archives with the same extension and archives with other extensions. But most of them have the same head CHNK. So if someone can get into the archive mentioned above we would also have access to most of the other ones. Then we would know if the format of 3D models can be edited with known software or not.

If you want the complete archive, please tell me and I'll send you a link via PM (file size over 700 Mb).

Here are the parts cutted with MultiEx Cutter Tool: http://www.ftp-driver-welten.com/privat ... les.sp.zip
