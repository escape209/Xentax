Post by aluigi » Mon Feb 09, 2009 4:53 pm

the decompilers can help a lot although don't expect to see the original code ready to use.
in general for the executables there are commercial solutions like hex ray or the free, and very good since I have used and still use it, REC: http://www.backerstreet.com/rec/rec.htm
there is also Boomerang (http://boomerang.sourceforge.net) but I have never had good results with it.

exist also decompilers written for a specific language (like the class files of java or the python compiled code) or for executables built with a specific programming language (like C# and .net)
