I don't know how the lump.dat file works but this seems to work for the archives.

Code: Select all

#Travia 2 archive unpacker
#Written by Tsukihime
#Feb 14, 2012

get lump_num long
get nameLen long
getdstring lump_name nameLen
get FILES short
for i = 0 < FILES 
	get unk short
	get unk long
	get len long
	getdstring NAME len
	get unk1 long
	get unk2 long
	get unk3 long
	get unk4 long
	get unk5 long
	get unk6 long
	get SIZE long
	savepos OFFSET
	log NAME OFFSET SIZE
	math OFFSET += SIZE
	goto OFFSET
next i
