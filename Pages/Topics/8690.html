Post by chrrox » Thu Apr 05, 2012 2:39 pm

podlzx files are plain pod files compressed with xbcompress.exe from the sdk.
you can decompress them with xbdecompress.exe

here is a pkg extractor for the pkg files.
for
set MAGIC 0
Do
getdstring MAGIC 4
While MAGIC != "adoY"
getdstring MAGIC2 4
if MAGIC2 == "oMoN"
cleanexit
endif
get SIZE long
getdstring NAME 0x54
savepos OFFSET
log NAME OFFSET SIZE
math offset + size
goto offset
Padding 16
next
