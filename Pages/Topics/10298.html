Post by Teancum » Sun Apr 07, 2013 1:49 pm

Recently a total of three sources were released:

http://jkhub.org/topic/1941-jedi-academ ... -released/

The original two, dubbed 'jedioutcast' and 'jediacademy' are actually Jedi Academy for PC, and Jedi Academy for Xbox, respectively. The original Jedi Outcast source was later uploaded. I'm excited mostly for the fact that the Xbox source includes the complete code for Vicarious Visions' GOB/GFC container format, which was used on several games like Jedi Outcast/Academy, Crash Nitro Kart, and Doom 3 (all console versions). Hopefully someone will be able to use the source to create an archive tool so I can create some console-based mods on these games.
