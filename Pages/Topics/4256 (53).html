Forza 3 Disc one autounpacker (Cars / Wheels)
http://www.mediafire.com/?5p5zf17v22f2uic

Extracts forza vehicles/wheels into seperate folders
Use wordpad - replace function to modify if necessary.

quickbms -o zip.bms c:\cars\ACU_15_ARX01b_08.zip "c:\Forza Extracted\ACU_15_ARX01b_08"
quickbms -o zip.bms c:\cars\ACU_26_ARX01b_08.zip "c:\Forza Extracted\ACU_26_ARX01b_08"
