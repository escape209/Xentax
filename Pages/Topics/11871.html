Post by Ekey » Wed Sep 03, 2014 6:27 am

Here script for unpack, but this archive don't contain filenames.

Code: Select all

# MegaRace 3 (BZF format)
# 
# Written by Ekey (h4x0r)
# 
# script for QuickBMS http://quickbms.aluigi.org

get FILES long

for i = 0 < FILES
    get OFFSET long
    get ZSIZE long
    get SIZE long
    get HASH long
    get NULL long
	
    if SIZE == ZSIZE
        log "" OFFSET SIZE
    else
        clog "" OFFSET ZSIZE SIZE
    endif
next i
