BMS script for your sample:

Code: Select all

## Hana to Taiyou to Ame to (PS2) bin extractor script by BloodRaynare

IDstring "LF"
get FILES short
for i = 0 < FILES
	get OFFSET long
	get SIZE long
	math OFFSET * 0x800
	string NAME p "FILE%04d." i
	log NAME OFFSET SIZE
next i
