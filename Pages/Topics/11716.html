If you guys want to unpack EDAT files from this game, you guys need to grab RAP file that contains RIF keys,

and EDAT can be easily decrypted with KLIC KEY and RAP file on these days,

try to find this file on the internet, "EP0002-NPEB02097_00-DSTPUBLICBETAP30.rap" this will do the trick to decrypt NPDRM EDAT files

Of course you guys need EDAT Decrypter as well, Jikkyu's EDAT Rebuilder 1.45 c works really well
