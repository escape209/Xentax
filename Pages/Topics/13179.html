Post by HiddenKnowledge » Mon Aug 10, 2015 11:54 pm

This questions seems to assume that we only need credit for products but sometimes we just want to discuss something related to it or only use it for personal projects.
Personally if I use something from any forum or website, I link back to the relevant post. If I (want to) use it in my product I contact the post author if there isn't a license for it already and ask them for a license/permission to use it.
If I use it for a personal project that will never see daylight, I use the relevant thanking feature (Post a reply/Press thanks) and if it happens a lot then I will let my friends know about the forum/site as well/post it on social media.
I really prefer the thanking option though because it means I don't need to bug everyone subscribed with my "thanks" post.

I voted yes because I try to give credit where possible.
