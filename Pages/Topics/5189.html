I was just having the same problem (and I'm gonna use the models for a 2d fighting game).
Download the MD5 to Max script from http://www.doom3world.org/phpbb2/viewtopic.php?t=3229 and run it inside max. Import the md5mesh with the script and you're done. (I'm not because I really don't know how to use max)
Export the model to 3ds(or whatever) then import it in Milkshape. delete all of the groups that don't start off with numbers(because these are those spikey meshes that are all over the character.) then reassign all of the groups with similar materials to one material. (because each group has it's own material with a title such as ssk_txt.tga; ssk_txt.tga1; ssk_txt.tga...) Once that's done, correct the name of the texture file, and I'm done .

Also do you actually mean Naruto Clash of Ninja 3 or Naruto Shippuden Clash of Ninja Revolution 3? Because if so, then can you tell me how, because I wanted to get the 3rd, sound 4, haku, and zabuza frorm that game.
