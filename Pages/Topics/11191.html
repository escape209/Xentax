Post by Darkstar » Thu Feb 06, 2014 3:02 pm

My basic understanding is that posting links to anything illegal is not allowed. That includes file sharing sites (torrent sites, ed2k sites, sites which link to downloads of copyrighted stuff on filehosters, etc.) and directly linking to copyrighted stuff (e.g. URLs to game data files on whatever file sharing or file hosting site you can imagine)

It is, however, allowed to link to file hosters for non-problematic content (i.e. everything that you created yourself, like source code, binaries, etc. where the copyright status is clear)

Post by Darkstar » Thu Feb 06, 2014 10:57 pm

Complete game data files are definitely a no-no. Sometimes uploading the first few hundred kb or so is silently tolerated (or goes unnoticed for a couple of days) but don't count on it. I usually paste the link as text-only in these cases, because that doesn't actually qualify as a "link" (you have to mark it and copy/paste to the address bar) but people interested can still get it. At least in our jurisdiction here there are some court decisions that seem to suggest that non-hyperlinked links are treated differently than "real" links
Check out the REWiki!
