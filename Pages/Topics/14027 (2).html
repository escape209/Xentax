Hi

We can use for each vertex 4 weights- the same [0.25,0.25,0.25,0.25].


about bone map

Code: Select all


file: baron.bmd6model;baseModel



Here is  a list of bone  indices

go to section SecId=6 ( http://forum.xentax.com/viewtopic.php?p=117917#p117917 )

offset 1328	(0, 1, 2, 5, 8, 11, 14, 27, 40, 49, 52, 55, 41, 42, 43, 44, 45, 46, 15, 28, 47, 50, 53, 56, 58, 66, 74, 59, 67, 75, 60, 68, 76, 61, 69, 77, 82, 16, 29, 48, 51, 54, 57, 62, 70, 78, 63, 71, 79, 64, 72, 80, 65, 73, 81, 83, 3, 6, 9, 12, 17, 30, 18, 31, 19, 32, 20, 33, 21, 34, 4, 7, 10, 13, 22, 35, 23, 36, 24, 37, 25, 38, 26, 39, 84, 85, 86, 87, 88, 89, 90, 91, 92, 93, 94, 95)

we must convert 
[0,1,2,5,8,11,14...] = [0:0,1:1,2:2,3:5,4:8,5:11,6:14...] - just add index to each element in list,  index:value

now we replace index=>value and value=>index like 5:11=>11:5 where 11 is now index and 5 is value

0:0,1:1,2:2,...,5:3,...,8:4,...,11:5,...,14:6...

it is correct bone map
[0, 1, 2, 56, 70, 3, 57, 71, 4, 58, 72, 5, 59, 73, 6, 18, 37, 60, 62, 64, 66, 68, 74, 76, 78, 80, 82, 7, 19, 38, 61, 63, 65, 67, 69, 75, 77, 79, 81, 83, 8, 12, 13, 14, 15, 16, 17, 20, 39, 9, 21, 40, 10, 22, 41, 11, 23, 42, 24, 27, 30, 33, 43, 46, 49, 52, 25, 28, 31, 34, 44, 47, 50, 53, 26, 29, 32, 35, 45, 48, 51, 54, 36, 55, 84, 85, 86, 87, 88, 89, 90, 91, 92, 93, 94, 95]
