Post by chrrox » Tue Sep 28, 2010 10:36 am

This is a bms script to extract the assets of Castlevania: Lords of Shadow
I tested this on the ps3 files.

Code: Select all


idstring BFPK
get version long
get files long
savepos TMP
for i = 0 < files
goto TMP
get NSIZE long
getdstring name NSIZE
get size long
get offset long
savepos TMP
goto offset
get zsize long
savepos OFFSET
if zsize == size
log name offset zsize
else
clog name offset zsize size
endif
goto TMP
next i

It still didn't work.

However this script did (took from Clive Barker's Jericho)

Code: Select all

ImpType Standard ;
Goto 8 0 ;
Get FNum Long 0 ;
For n = 1 to FNum ;
Get FNLen Long 0 ;
GetDString FN FNLen 0 ;
SavePos FOO 0 ;
Get FO Long 0 ;
SavePos FSO 0 ;
Get FS Long 0 ;
Log FN FS FO FSO FOO ;
Next n ;
