Script for QuickBMS:

Code: Select all

IDString "STF\x00"

get unknown_1 long
get unknown_2 long
get unknown_3 long # Count of sub-folders?

get fileCount long

get dataBlockSize long

get infoBlockSize long

for I = 1 to fileCount
get ff byte
get nameLen short
GetDString fName nameLen
get isFolder byte

if isFolder == 1
math I - 1
get objCount long
else 
get fSize long
get fOffset long
math fOffset + 9  
log fName fOffset fSize
endif

next I
