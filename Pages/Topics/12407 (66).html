Figured I should post this since it's noteworthy imo, and helpful for anyone modding MGSV: thanks to EcheloCross the MGSV Dictionary Project has got an additional 6284 filenames! (in addition to the 1645 filenames they also found a few weeks before!)

A lot more texture files are named now, so this should be especially helpful for texture modders.

To use this dictionary simply replace the dictionary.txt/qar_dictionary.txt in your QAR tool of choice with the dictionary.txt from the github.
