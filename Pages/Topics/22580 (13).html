So just a couple of things...

Could someone confirm this workflow please? (I only need working SMD skeleton animations, I do not need a skinned mesh to be connected)

1. Use Neosis to extract the psarc files from the Disc Images (we are all using D****X I assume and the PKG extraction code is all zeros?)
2. From these extracted folders and files find the Skel .paks and the Anim .paks
3. Use the tlou2.exe tool via cmd with "tlou2.exe ellie-skel.pak /rs" (to extract just the skeleton) (making sure any anim .paks are in the same folder)
4. Now drop the anim.pak (Should be related to the skeleton) onto the tlou2.exe to extract all the anims (lots of smd files)

Lastly, has anyone found any of the cutscene animations? This may not even be a thing as they sort of merge into them from gameplay and it may be a case that only certain "in-game" animations are available and the heirachy of the folders. Could anyone confirm this?

I would love to be able to sync up cutscene animations but I don't know if they've been found yet.

Thanks and thank you for the tool.
