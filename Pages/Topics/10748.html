Post by Ekey » Wed Sep 04, 2013 11:46 pm

Here script for PAK files.. Files seems obfuscated / xored

Code: Select all

# Manhunt (PS2) (PAK format)
# 
# Written by Ekey (h4x0r)
# http://forum.xentax.com
# 
# script for QuickBMS http://quickbms.aluigi.org

idstring "MHPK"
goto 0x8
get FILES long

for i = 0 < FILES
	getdstring NAME 260
	get SIZE long
	get OFFSET long
	get TYPE long
	get CRC32 long
	log NAME OFFSET SIZE
next i
