I'm running into the same issue FinalBlast is having, but only with files that have to do with dialogue and only to a certain extent. Final, the file you uploaded has dialogue in it which seems to be why it doesn't have the proper start to a streaming audio file like Alpha said. However, the correct audio files (Pure BGM) can be played back and converted just not in a traditional way. You need an Audio Editor like Audacity or Goldwave then import the file as raw audio. The specifications for said import are auto-detected correctly if you have a pure BGM file, if the import tool says its properties are anything other than signed 16-bit PCM, you don't have the right type of file. And now a few side-notes.

1) Final, if you have any further troubles either PM me or just reply to the topic.
