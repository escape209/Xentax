Sorry to bump an old post but this thread comes up first when researching this format so thought it would be the best place to put it. I'm working on a private server code for SBOL and although the textures and models were low priority the format was quite simple so thought I'd make a utility to edit and export them. The format essentially stores ARGB pixel data compressed in RLE.

I've uploaded the source to github of a working MIA texture file manager. https://github.com/tofuman0/MIA-Manager
I'm not great with 3D model formats but will take a look to see if I can work out the format at least.

I'm not sure if this format is used in any other games.

(BTW the SBOL code on my github is extremely old so don't bother using it. I plan to upload the latest once I have NPC's and other player packets working)
