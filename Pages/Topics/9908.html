Post by MrAdults » Sat Nov 24, 2012 9:09 pm

I've created a new SVN repository where chrrox, revelation, finale00, demonsangel, and I can put all of our Noesis script and plugin stuff. It has Python scripts and binary plugins. demonsangel, chrrox, and I have already started adding stuff. This is where you can get the latest versions of all of our stuff from now on. (the stuff that isn't included in the actual Noesis distributions, that is)

https://code.google.com/p/noesis-plugins-official/

If you would like to be added as a plugin/script writer, send me a private message and point me to some of your work. Once the repository is a bit more full, I'll probably start occasionally packaging it up and adding snapshot distributions to the downloads section.
