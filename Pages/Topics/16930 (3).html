PrinceMyles wrote: ↑Thu Apr 20, 2023 8:14 am Hey guys quick question : I am trying to convert the dae.phyre from hajime No Ippo : The Fighting.
Do you know wich exe I have to use for that convertion back to .dae (if there is any existing) I found the texture script for Noesis but all the .exe here doesn't look like they work for that specific PS3 game.

Kind regards!
It's pretty simple to get the models. Go to USRDIR -> Data -> GCM -> Characters which has a folder for each of the characters in the game. Go to the models subfolder and run kiseki2.exe on them, you'll get a useable .smd which you can open in Blender by downloading the Source modelling plugin that opens smd's. This won't work with the animation files though which are also .dae.phyre... hope someone can figure out how to get the animations working on the models (which have useable skeletons)....
