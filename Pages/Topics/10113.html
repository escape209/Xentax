Post by Gh0stBlade » Sat Mar 01, 2014 5:52 pm

Code: Select all

# The Mighty Quest For Epic Loot [.BFPC]
# Script v1.0
# By: Gh0stblade

goto 0x100

get chunkSize long
get dummy long
get numChunks long

for i = 0 < numChunks
get name basename
string name += "_"
string name += i
string name += ".bin"

set offset i
math offset *= chunkSize
math offset += 0x800 
log name offset chunkSize 
next i
This should split out those chunks.
