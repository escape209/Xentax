Here tool for decrypt and encrypt back BIN files. Output decrypted files just simple text.

Example using:

Decrypt

Code: Select all

VBABinTool "data\text\English\Globals.bin" "data\text\English\Globals.txt"
Encrypt

Code: Select all

VBABinTool "data\text\English\Globals.txt" "data\text\English\Globals.bin"
