Here it is - unpacker/packer for base Deathspank game

Code: Select all

http://raptor.cestiny.cz/download/deathspank-gg-files-extractor.html
Open original gg file, select "Patch" button, enter new filename, select folder with modified files (only modified files) and wait. Save file with new filename. Overwrite original file is not possible now - because of data streaming. This tool si NOT for DeathSpank TOV. Wait for new one. TOV has different data files.

Page is not in english for now, sorry. But you will find out download link

Use it, test it, enjoy it.
