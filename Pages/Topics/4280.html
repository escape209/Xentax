I think I'm going to resurrect this thread

I've created a program to decode the audio from this game (I like to call the codec 'TSubBand'). You can download it at this link. I can't say it's fully tested, but it should work. After extracting the files from the .erb resource, run it with:

Code: Select all

DecodeTSubBand InputFilename.wav.TSubBandResource.bin OutputFilename.wav
