Okay, yeah, sounds like you're downloading the Japanese version's DLC files with the English voices built-in instead (which also explains the larger file size). That one seems to have a different key (which is why you're having trouble with it), but fear not as those those aren't actually necessary, since all of the content there is included with the US version and its update.

The one that you're looking for should have only three parts, and has the ID "UP0082-CUSA12031_00-KINGDOMHEARTSX30-A0110-V0100". That one should work with the key, and the following PAKs should have the Re:Mind DLC models and such:

Code: Select all

pakchunk0-ps4_1_p.pak
pakchunk0-ps4_2_p.pak
pakchunk0-ps4_3_p.pak
pakchunk0-ps4_4_p.pak
pakchunk0-ps4_5_p.pak
pakchunk0_s1-ps4_3_p.pak
pakchunk0_s2-ps4_3_p.pak
pakchunk0_s3-ps4_3_p.pak
