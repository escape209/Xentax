Post by Keygen » Sat Aug 29, 2015 10:01 am

Hello again,

Finally i found the way to extract correctly those files.

Isnt using offzip, i used QuickBMS with this script:
---------------
#quickbms script
#from chrrox and aluigi
open FDDE VFS 0
open FDDE INF 1

get id longlong 1
get files long 1
goto 0x48 1
for i = 0 < files
getdstring name 0xD8 1
get ZIPPED long 1
get DUMMY long 1
get offset longlong 1
get zsize longlong 1
get size longlong 1
get TIMESTAMP longlong 1
get TIMESTAMP longlong 1
get TIMESTAMP longlong 1
if ZIPPED == 0
log name offset zsize
else
clog name offset zsize size
endif
next i
---------------
