Post by chrrox » Fri Jun 17, 2011 7:26 pm

quickbms script to extract these.
make sure you use the ini decryptor first.

Code: Select all

#quickbms script
#Archlord PC
#from chrrox

open FDSE Data.Dat 0
open FDSE Reference.Dat 1


get files long 1
get foldersz long 1
get folder string 1
string folder + \
for i = 0 < files
    set name folder
    get NAMESIZE long 1
    getdstring name1 NAMESIZE 1
    string name + name1
    get offset long 1
    get size long 1
        log name offset size
next i
