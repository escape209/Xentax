Post by zaramot » Sat Jun 30, 2018 9:33 pm

Unique NPC like Arcann, Valkorion, Darth Malgus ect. are easy to find, as for other NPC's like Darth Marr, Thanaton and so on, I suggest you to use Jedipedia (there you could check which parts NPC is using) and then just find them by it's names, it's easy actually.
Like this for example https://swtor.jedipedia.net/en/npc/lord-scourge-2
For chest it's using chest_capetight_bms_heavy_sw_a06 parts, just use windows "find" function in folrders where you extarcted files.
