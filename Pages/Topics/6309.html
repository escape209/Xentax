Post by WRS » Wed Aug 15, 2012 8:39 pm

completely untested. based off source from sau-0.2

Code: Select all

# -- header
get nfiles short

# -- files
for f = 0 < nfiles
  getdstring Name 14
  get Off long
  get U1 long
  putarray 0 f Off
next f

goto 2
math nfiles -= 1 # the last file entry just stores eof apparently

for f = 0 < nfiles
  getdstring Name 14
  get Off long
  get U1 long
  math ff = f
  math ff += 1
  getarray Size 0 ff
  math Size -= Off
  log Name Off Size
next f
edit

the project also includes some code for the graphics/sprites. here is the source for anyone looking it up: http://pastebin.com/WrzmvWbV
