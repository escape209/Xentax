Post by aluigi » Wed Apr 20, 2011 7:51 am

are you sure that the file you uploaded is complete?
because it looks that the last 8 files are not available.

the following is the quickbms script that will automatically skip the unexistent files:

Code: Select all

get DUMMY long
get DUMMY long
get FILES long
get BASE_OFF long
get DUMMY long
goto 0x40
get LAME asize
for i = 0 < FILES
    getdstring NAME 0x38
    get OFFSET long
    get SIZE long
    math TMP = OFFSET
    math TMP += SIZE
    if TMP <= LAME
        log NAME OFFSET SIZE
    endif
next i
