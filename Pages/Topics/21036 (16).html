ikskoks wrote: ↑Thu Apr 21, 2022 11:19 pm Reimporting process is similar to exporting.
You basically do these steps:
1. Click on "reimport.bat" in quickbms.
2. Select BMS script.
3, Select "Input archive". In your case it will be original PTE texture file.
4. Select output directory. In your case it is a directory where CRN file was extracted (e.g. "output1").

In case with further issues with quickbms, try to find a tutorial to guide you through the process
