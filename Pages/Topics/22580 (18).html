sorry for inconvenience ..
make a new empty folder, copy ellie-skel in it and extract "\>tlou2 ellie-skel.pak /r", so you have the smd file called "ellie-skel_skel.ellie-skel" and a new subfolder with other files, ignore that subfolder first. Now you have the skel of ellie, if you open it i blender with smd plug in you can see it. Now you need the body parts for the skel. copy in your folder for example ellie-head and extract it. Now you open the "ellie-skel_skel.ellie-skel" in wordpad or in anay writing program. So you have to see:
version 1
nodes
0 "root" -1 .... etc

scroll it to the end. Now you open the ellie-head.smd file in wordpad too. copy the hole text and copy it to the end of the ellie skel file in wordpad. Than save it as "smd". If you now open it in blender you have to see Ellies skel with the head. This if you want the hole body you have to do it with all body parts copy it in one file together and save it as smd.

Now the animation:
copy for example any ellie animation for example "anim-ellie-mm-melee" and copy it in the first folder where you extract the skel drop it on the tool and a new subfolder will write with the animtion it it. So open your amd skel with your body parts file in blender, and import the animtion smd file there, and you will see your ellie do the animation you choose. sry for my bad english hope i could help you
