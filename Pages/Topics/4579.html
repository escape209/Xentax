NBA 2K10 and it excellent models.





Game folder files list attached below.
I very interested in converting models, weight data and textures from this game (and similar-engine game to).

Game file extension:
CDF - some kind of storage for IFF files (95 files, 512mb)
IFF - scripts, textures, model data (3747 files, 3.9gb)
BNK - animaudio00X.bnk, where X is number from 0 to 7 (8 files, 5.44mb)
BIN - archive, i don't know for what (31 files, 1.88gb)

Ok. In details:

There is 90 IFF files with name:

pchead_XX.iff
pchead_XX_hb.iff

where XX is number form 00 to 44. I think it's model of heads. Here some of them.
Also near of 'pchead' have files:

pcface.cdf (3 431kb) download
pcface.iff (1kb)
pcface_lo.cdf (46 297kb)
pcface_lo.iff (4kb)
pcfacehair.cdf (31 984kb)
pcfacehair.iff (25kb)
pchair.cdf (5 063kb)
pchair.iff (2kb)

and

pcmorphs.cdf (23 547kb) maybe face morph settings for chunk of head (pchead).
pcmorphs.iff (4kb)

Next we have XdornaXXX.iff name-like files. In this post answer what is this, but i don't understand what is this actually. Anyway here the samples.

There many pngXXX.iff (894pc, textures) and coachXXX.iff (157pc, for models and textures of coaches i think) files, where XXX is number. PNG samples. COACH samples.

Here samples of other files. With the name like r1h020.iff or s550_lo.iff.

Also have files:

paplayers.bin (80 360kb) download
pcchest.cdf (3 825kb) download
pcchest.iff (1kb) download
players.bin (80 145kb) download

So will be nice convert this game files in dds/png/tga and dae/smd/fbx.
