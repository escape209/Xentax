3ds Ram Dump

  • Report
  • Quote
  • Thanks

Post by chrrox » Tue Jun 24, 2014 2:35 pm

This is a guide for current 3ds ram dumping method.

First you need a 3ds with firmware <= 4.5

2nd you need a 3ds compatible ds flash cart or a friend with one to install
the initial gateway loader.

After you install the gateway loader you need to put
fcramdump
on your normal sdhc flash card.

Now you can only dump retail carts that are compatible with your region 3ds.
Load up your game and get to the point where you want to dump ram from.

Then go into the system menu and launch the ds profile exploit and it will dump the game ram to your sd card.

The dump should be around 130mb.

If anyone has ram dumps they want me to help with let me know.

This can be used to dump audio and model data.
