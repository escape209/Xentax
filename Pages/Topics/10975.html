lumekano wrote:thax for Script. and i'm srry about noob. how to you extract arc file FolkLore PS3?
quickbms script

Code: Select all

endian big
open FDDE fht_ 0
open FDDE bin_ 1
get files asize
math files / 0x28
for i = 0 < files
getdstring name 0x20
get size long
get offset long
log name offset size 1
next i
