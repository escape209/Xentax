Sorry for the delay.
You need to make a ISO image (ISO format exactly), then find a program called Apache 2, with it you should open the iso and look for an option to get the Layer2. With it you can make another iso of the layer2, then you can extract the files.

BTW you should know that the disk is double layered since Windows only reads the first layer, but ISO programs (I used DVD Decrypter I remember) makes ISOs larger than a single layer disk size.
