Post by daemon1 » Mon May 11, 2020 9:01 am

ophy111 wrote: ↑Mon May 11, 2020 12:08 am 130kb from doing only bt/uasset/materials. there were no errors in cmd as far as I could tell..
This is fine.

For map extraction:
- you DONT need BaseMaterials and BaseMaterialInstance folders
- you DONT need any textures
- you DONT need staticmesh Folder

All you need is:
1. kh3.exe
2. material.db file
3. staticmesh.raw folder
4. umap+uexp of a map you want to extract

It DOESNT matter where (in which folder) all these files are.

Somebody asked me to make it possible that source files will be in one folder, and extracted map go to some other folder, and i made it possible. But now, to make it simple, lets forget about it. For sure, you have to place all needed files (listed 1,2,3,4 above) to one folder, and then drop umap onto EXE or run EXE from command line with 1 parameter - umap file.

This must work. If it doesnt, send me output text or screenshot.
