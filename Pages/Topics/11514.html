I made a script do extract the files but they seem to be compressed or encrypted.
If someone could help with this that would be great.
A bit of a messy bms script but it extracts lol

Code: Select all

endian BIG
open FDDE ARC 0
open FDDE BIN 1
get null1 long 1
get null2 long 1
get files long 1
goto 0x10 1
get fileoff long 1
goto fileoff 1
savepos ntablestart1 1
goto 0x20 1
savepos ntablestart2 1
for i = 0 < files
goto ntablestart2 1
get nameoff long 1
get size long 1
get unk2 long 1
get unk3 long 1
savepos ntablestart2 1
goto ntablestart1 1
get offset long 1
savepos ntablestart1 1
goto nameoff 1
get name string 1
log name offset size 0
next i
