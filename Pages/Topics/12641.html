Hi
Game: http://gjqt.gamebar.com/gj2column/download.shtml
tool for zpkg files: rpgviewer

Here is importer for models and animations.
It requires Blender version 249b and Python 26.
How to use:
1.run Blender249.blend
2.in Blender Text Window press alt+p and select:
-xac - for import textured and rigged models
-xsm - for import animation

Updated 2015-03-02:
- fixed animation for big endian
