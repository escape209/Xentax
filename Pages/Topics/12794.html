Post by barti » Fri Apr 24, 2015 5:25 pm

It's BPE (byte pair encoding).

QuickBMS script:

Code: Select all

get NAME basename
string NAME += "_unpacked.dat"
comtype bpe

get SIZE long
get ZSIZE long

clog NAME 8 ZSIZE SIZE
