So I think that this works as an extractor and repacker: https://mega.co.nz/#!XdJQ0CDQ!ZY9ptbqLM ... sxtyYV0vA8. To use it simply drag and drop the files in the executable. Make a backup of the original .bin file before extracting.

I've done some tests but not exhaustive ones, so if you notice any error, please contact me.
