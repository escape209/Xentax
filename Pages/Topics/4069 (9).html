1) Use this plugin, not the Autodesk's piece of rotten shit: http://opencollada.org/download.html
2) After importing the model into Max, apply the UVW Xform under the skin modifier, flip the V Tile and collapse to the UVW Xform. This will fix the texture coordinates.

revelation wrote:The 360 file is named differently than the ps3 files, as such does not match with the way the original script file is attempting to match the filelist with the data bin files. rename them to work with the script as is, or modify the script to look for the names correctly.

ie. rename filelistu.x360.bin to filelist_imgu.x360.bin

Hope that helps.
Use quickbms and the ff13.bms script and follow the above tips.
