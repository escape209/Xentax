Here's the file structure :

Code: Select all

{
	?header[4b] {
		0x00000000
	}
	files_in_archive[4b] {
		0x00000038
	}
	structure {
		row[0] {
			begin_of_file[4b]{
				0x00001DC8
			}
			size_of_file[4b]{
				0x000002F6
			}
			file_name[112b] {
				LVUP1.TXT // UNICODE!
			}
		}
		...
		row[n] {
			...
		}
	}
	data {
		...
	}
}
P.S.
These TXT's are actualy UTF-16 encoded, so keep that in mind.
