I did a little research of this game's files.

The game uses many SPR files - some kind of archives.
Those archives are not compressed, as far as I could tell.
The first file in each archive is not encrypted
(Like the first BIK file in video.spr - the intro movie. You can extract it from the archive by copying it out with a HEX edtior).
The other files in the archives are somehow encrypted.

I hope it helps you. If anyone finds anything interesting, tell me.
