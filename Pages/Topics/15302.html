It works in Blender 2.79b.

Open Blender 2.79b, go to User Preferences, Add-ons, and type 'rip' into the search box.
Once you located the installed NinjaRipper add-on, you should see the following descriptions:
- Description: Import RIP files
- Location: File > Import
- Author: Dummiesman
- Version: 0.0.1

If you have different Ninja Ripper add-ons installed, enable one and disable the others.
