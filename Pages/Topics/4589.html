V9 beta 4 has just been released on my site. http://sf4mods.blogspot.com

- Editing of catalog.dlc files to allow for creating new dlc
- Misc.Pac editor for WWE 13 and 2k14 formats
- Zlib functionality (no arc file updating)
- Support for pac files containing directories (Renders, Chetc, etc)
- Changes file headers for EPK8 function
- Adding new entries to arc files for new pac files added to the game
- Uncompressing all files in a pach file in one go
- Caw injector (xbox only)


WWE 13 Specific Functionality


- Editing of wrestler info (ps3 and xbox)
- Caw injector (ps3 and xbox)
- Entrance editor (ps3 and xbox)
- Moveset export and import (ps3 and xbox)


WWE 2k14 Specific Functionality (Xbox only)
- Editing of wrestler info
- Entrance editor
- Moveset export and import
