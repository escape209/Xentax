Update 2014-09-14:
- C9 importer for Blender 249 (not work with newer version)
- install Blender 249 and Python 26
- unpack archive
- click Blender249.blend
- press alt+p in Text Window
- select:
r3cb for skeleton
r3cm for skinned mesh (script get all *.r3cm files from folder)
r3ce for animation

Work with game from 2013.
