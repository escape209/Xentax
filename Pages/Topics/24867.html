Bigchillghost wrote: ↑Wed Dec 29, 2021 12:44 pm
Yep, Ekey made a script for the .e3p, but these are .e3m, gonna attach the script hope it only need some tweaks .

Code: Select all

# SUN WUKONG (E3P) format (script made by Ekey)
# script for QuickBMS http://quickbms.aluigi.org

comtype lzma86head
get MAX_OFFSET asize

do
   get NSIZE long
   getdstring NAME NSIZE
   
   set EXT extension NAME

   get ZSIZE long
   savepos OFFSET
   math TEMP = OFFSET
   math TEMP += 5
   goto TEMP
   get SIZE long
   
   if EXT == "ATF"
       log NAME OFFSET ZSIZE
   else
       clog NAME OFFSET ZSIZE SIZE
   endif

   math OFFSET += ZSIZE
   goto OFFSET
while OFFSET < MAX_OFFSET
