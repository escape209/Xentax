Post by RENIKRILL » Tue Aug 21, 2012 9:52 am

I wasn't able to figure out if Paris is burning or not, but here is a script for quickbms to extract those archives, as also requested.

Code: Select all

idstring "HAB0"
get UNKNOWN short	# 1
get UNKNOWN short	# 2
get TOTALSIZE long
get NULL long
get UNKNOWN short	# 0x10 - length of each entry...
get UNKNOWN short	# 5
get FILES short
get UNKNOWN short	# 0x10 - length of each entry...
get FNDIRSIZE long
get BLAAAAAAA long
math FNDIR == FILES
math FNDIR *= 16
math FNDIR += 32
log MEMORY_FILE FNDIR FNDIRSIZE
math DATAOFFSET == FNDIR
math DATAOFFSET += FNDIRSIZE

for i = 0 < FILES
	get FNPOS long
	get OFFSET long
	math OFFSET += DATAOFFSET
	get SIZE long
	get NULL long
	goto FNPOS MEMORY_FILE
	get NAME string MEMORY_FILE
	
	log NAME OFFSET SIZE
next i
I only had a look at one of the archives (DATA.HAB), but I'm confident it should work with the other of the games "HAB" files.
Having said that; dont hesitate to let me know if a problem arises with any of the others, or if you simply wan't to tell me I'm a douche (I don't mind that, either )

Hey buddy.
I went for a yonder into 'BGMPACK.SRB'... They've (the devs) constructed this archive so that it begins similar in layout to the former 'HAB' archives, and then impromptu goes off on an acid-trip; scattering random bytes of seemingly-meaningless value, whilst omitting pieces of playback-information (which weren't essentially vital anyway, but would have been helpful).
As a result of this craziness (putting a single archive within another, completely different type of archive was just the precursor), I've ended up with a pretty hap-hazard sheet of script, so you (and of course, others too) will have to excuse my annotations and other obvious mess which I simply didn't bother to clean up before posting. whatever, sega.

math DIR2 *= 16
math DIR2 += 32
math DIR2SIZE == DIR2
math DIR2SIZE -= 32
log MEMORY_FILE DIR2 DIR2SIZE

for i = 0 < FILES
	get UNK short	# 4
	get STREAM_TYPE short	# 0x30 (w/ initial header) / 0x10 (headerless)
	get OFFSET long
	get SIZE long
	get UNK short	# 4
	get FREQUENCY short
	get UNK short MEMORY_FILE	# 4
	get UNK short MEMORY_FILE	# NULL
	get FILENUMBER short MEMORY_FILE	# ascending from 0
	get UNK short MEMORY_FILE	# 0xffff
	get NULL long MEMORY_FILE
	get SIZE2 long MEMORY_FILE
	if SIZE2 != SIZE
		print "SIZE difference!"
		cleanexit
	endif
	
	if STREAM_TYPE == 0x30
		log MEMORY_FILE2 OFFSET 16
		get STRHEADERSIZE long MEMORY_FILE2
		get UNKNOWN1 long MEMORY_FILE2	# loop-start (samples)??
		get UNKNOWN2 long MEMORY_FILE2	# half-way point (bytes)
		get UNKNOWN3 long MEMORY_FILE2	# 0xFFFFFFFF -- loop-end??
		if STRHEADERSIZE != 16
			print "wrong STRHEADERSIZE: %STRHEADERSIZE%"
			cleanexit
		endif
		if UNKNOWN3 != 0xFFFFFFFF
			print "problem with UNKNOWN3: %UNKNOWN3%"
			cleanexit
		endif
		math OFFSET += 16
		math SIZE -= 16
	elif STREAM_TYPE == 0x10
		# do nothing here
	else
		print "unknown STREAM_TYPE: %STREAM_TYPE%"
		cleanexit
	endif
	set CHANNELS 2
	set INTERLEAVE 0x2000
	
	set MEMORY_FILE3 binary
