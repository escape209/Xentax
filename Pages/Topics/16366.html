Edit texture: use ARC TOOL or hex editor to convert a dds file (dont change file size) & (dont chang type of dds: dxt1, dxt3, dxt5)
Edit font file (gfd): we dont have any tool for .gfd file... Try on hex editor
Edit scripts: use hex editor
