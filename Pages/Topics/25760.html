spamzilla wrote: ↑Thu Sep 01, 2022 4:18 pm What is a model sample??
Hahaha, good one. What's a model sample? That's a model asset which contains the vertices and faces (at least) of a 3D model as binary data.
There's should be a bres (or BRES) signature/texture string in those WII 3D samples (it's a WII game, isn't it?).

Usually the model files are packed in a (mostly compressed) archive, though.
