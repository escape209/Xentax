You don't need repack files.

Extract gamecrysis2/localized/yourlanguage.pak, you will get a languages folder with the extracted files, then move this to the gamecrysis2 folder.
Extracted (and edited) files will override the pak files - I tested this method with the text / audio files and it's working.

quickbmsver 0.4.10b # correct reimporting granted from this version
set CRYSIS2_KEY binary "\xc9\x68\xfb\x67\x8f\x9b\x42\x67\x85\x39\x9e\x84\xf9\xb9\x9d\xc4"   # retail

# XTEA (ecb) decryption
