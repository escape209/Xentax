Post by Karpati » Tue Apr 14, 2020 6:23 pm

You can open the Hellgate London .wm files usign the following programs:

- 3D Object Converter (Windows)
- i3DConverter (macOS)
- i3DConverter (Linux)

How to get the 3D Object Converter:
Download the 3D Object Converter from http://3doc.i3dconverter.com and install it or download and use the portable version.
After it just use the Help/Check for updates... function to get the v7.027.

How to get the i3DConverter macOS:
Download the i3DConverter from http://www.i3dconverter.com and install it.
After it just use the Help/Check for updates... function to get the v3.704.

How to get the i3DConverter Linux:
Download the i3DConverter from http://www.i3dconverter.com and install it.
After it just use the Help/Check for updates... function to get the v1.704.
