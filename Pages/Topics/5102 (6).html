Lorian Elder Prince wrote:So here are some samples, if someone could tell me what I'm supposed to do to fix them. Thanks in advance.
http://www.mediafire.com/file/vs9bk0av2 ... in+dds.zip
run this bms script on your samples to decompress them.

Code: Select all

# script for QuickBMS http://aluigi.altervista.org/quickbms.htm

comtype zlib_dynamic
get NAME basename
string NAME + _decmp.dds
get ZSIZE asize
math ZSIZE - 4
clog NAME 0x4 ZSIZE ZSIZE
