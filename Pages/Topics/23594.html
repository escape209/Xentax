I have finished my Muhon / Wu Hun .msh loader module and I have released the following programs as web updates:

- 3D Object Converter v8.017 (Windows)
- i3DConverter v4.107 (macOS)
- i3DConverter v2.107 (Linux)

How to get the 3D Object Converter v8.017:
Download the 3D Object Converter from http://3doc.i3dconverter.com and install it or download and use the portable version.
After it just use the Help/Check for updates... function to get the v8.017.

How to get the i3DConverter macOS v4.105:
Download the i3DConverter from http://www.i3dconverter.com and install it.
After it just use the Help/Check for updates... function to get the v4.105.

How to get the i3DConverter Linux v2.105:
Download the i3DConverter from http://www.i3dconverter.com and install it.
After it just use the Help/Check for updates... function to get the v2.105.
