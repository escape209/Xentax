As mentioned, you can run gitmo (the link still works above) for God Eater Burst. It doesn't really matter since the files in the RDP only have hashes to represent them, so you won't be getting original filenames either way. You can run gitmo on the iso or the rdp. Noesis views the GMO and GIM files extracted by gitmo.
