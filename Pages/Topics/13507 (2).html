daemon1 wrote:
pechenko121 wrote:
daemon1 wrote:I used Luigi's igz extract script
Where can I get this script?

Code: Select all

endian big
idstring "IGZ\x01"
get FILES long
for i = 0 < FILES
    get DUMMY long
    get NAME_OFF long
    get OFFSET long
    get SIZE long
    log "" OFFSET SIZE
next i
