https://www.dropbox.com/s/vr54uf2o3a41b ... ck.7z?dl=0 (1.2MB)

The pack contain:
- QuickBMS + bgeHD script (both created by aluigi)
- my tools (BGE02, BGECount), at least .Net 2.0 required
- unp2bin - bin repacker, created by CHEATER
- English files, unpacked English files and extracted texts
- Readme (Read this before you start!)

The PS3 version probably use almost the same files, but the PS3 .bf file is a mess, I can't unpack it correctly.
