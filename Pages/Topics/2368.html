Post by Mr.Mouse » Sat Jan 13, 2007 1:21 pm

Code: Select all

Get FNum Long 0 ;
Set OFF Long FNum ;
Set PO Long 0 ;
Set PS Long 0 ;
Math OFF *= 32 ;
Math OFF += 16 ;
Get D1 Long 0 ;
Get D2 Long 0 ;
Get D3 Long 0 ;
Set FO Long OFF ;
For T = 1 To FNum ;
SavePos FOO 0 ;
Get ENDOFFSET Long 0 ;
Set FileSize Long ENDOFFSET ;
Math FileSize -= PS ;
GetDString FileName 28 0 ;
Log FileName FO FileSize FOO 0 ;
Set PO Long FO ;
Set PS Long ENDOFFSET ;
Math FO += FileSize ;
Next T ;
Ok, this will open the start.dat. Just copy this into a text file and use that in the Run Mexscript on.. option.
