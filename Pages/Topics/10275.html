Here a quickbms script for unpack

Code: Select all

get files short
set table files
math table *= 16
math table += 2
for i = 1 to files
get offset long
math offset += table
get size long
getdstring ext 8
set name i
string name += "."
string name += ext
log name offset size
next i
I think there are no names in the archive, only extensions.
