Franc23 wrote:Is there a way to change to make it work?
If, and only if, the file contains "system" at its file offset 0x000010 you can cut the first 50 MB block from your 600 MB file using a hex editor: start offset 0x000000 to 0x2FAF080, size: 50000000 bytes
and try to load it into make_obj.
Tuts: a) Bigchillghost, viewtopic.php?f=29&t=17889
b) Extracting simple models: http://forum.xentax.com/viewtopic.php?f=29&t=10894
