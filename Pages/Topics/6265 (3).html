Neo Andersen wrote:Hi everyone! help with the packing .pak files after unpacking
Example

You simply drag&drop the need files you want to import.

You drag all steps
1.quickbms.exe and add -r -w
2.then the crysis2.bms
3.then the decompressed files from what pak file you want.After you drag it onto the cmd, add .pak to its end (its the extension you want that folder to be compressed to)
4.Then you drag the Crysis 2 folder where the original pak file is located.
5.Press enter, and the original pak file will be overwritten with whatever you wanted to reimport back to the original pak.
