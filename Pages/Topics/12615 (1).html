Escope12 wrote:
Mr.Mouse wrote:Does what work with other formats?
Does MultiEx Command & Mexscripts work with other formats
Of course, that's the whole point of MexScript and MultiEx when I created it back in 1997 . Check this old list of supported formats in MultiEx Commander:
http://multiex.xentax.com/suplis.txt

Read the manual here:
http://www.xentax.com/downloads/multiex ... Manual.pdf

Get the program for free here: http://multiex.xentax.com/

You can create a stand-alone mod for a game this way: http://www.youtube.com/watch?v=7dUN8C8Z3sE

Read the Mexscript guide here: http://wiki.xentax.com/index.php?title=MexScript

The MexScript will help you support new formats in MultiEx Commander. Just check the Let's MultiEx tutorials on how to proceed.

You may also want to read up on this: http://wiki.xentax.com/index.php?title=DGTEFF

7 Sins, by YouTube request!

Code: Select all

#7 Sins .PAK
IDString 0 "PACK" ;
ImpType StandardTail ;
SavePos TailOffOff 0;
Get TailOff Long 0 ;
SavePos TailSizeOff 0 ;
Get TailSize Long 0 ;
Set ArSize Long TailOff ;
Math ArSize += TailSize ;
Math TailOff += 160 ;
GoTo TailOff 0 ;
Do ;
SavePos CP 0 ;
Get FileName String 0 ;
Math CP += 72 ;
GoTo CP 0 ;
SavePos FOO 0 ;
Get FO Long 0;
SavePos FSO 0 ;
Get FS Long 0 ;
Log FileName FO FS FOO FSO ;
SavePos EB 0 ;
While EB < ASize ;
