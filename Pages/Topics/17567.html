Post by Acewell » Thu Feb 01, 2018 6:51 pm

this bms script will split your bin sample

Code: Select all

# script for QuickBMS http://aluigi.altervista.org/quickbms.htm

get BIN_SIZE asize
get FOLDER basename
math i = 1 
for i
	get OFFSET long
	savepos TMP
	if OFFSET == BIN_SIZE
		break
	endif
	get NEXT_OFFSET long
	xmath SIZE "NEXT_OFFSET - OFFSET" 
	string NAME p "%s\%d.dat" FOLDER i
	log NAME OFFSET SIZE
	goto TMP
next i
i can't help you with the contents of PS2 files but the last one looks like paletted image data.
