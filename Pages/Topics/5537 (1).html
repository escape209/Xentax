K, fixed up my old script for the human head demo, so that it would import from the other NMBs.
this time instead of grabbing the text names in the NMB, I scaned for the ID tags for each section.

I've never dealt with this sort of ID driven format before, and I'm a n00b to scripting/programming. so no surprise the script is very slow on import.

unfortunately the madussa NMBs are alot different then the fore-mentioned NMBs.. different variable sizes, and alot of extra text information for materials. also I saw alot of weird padding through out the file :\

Here's some screenshots:








I really love the Luna suit, that effect is pretty much pulled off by a single mask texture O_O



also the nalu model is the same model that nvidia used in dawn and dusk.. that model on its own seems to have been used in other 3d softwares.
most notably a game called "dreamstrippers" not sure where the dawn model first originated, but its been used in other things too.
