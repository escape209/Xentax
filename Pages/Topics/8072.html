Post by Axolotl » Wed Jan 18, 2012 4:07 pm

In later versions of Atlantica they changed some things. There were added a lot of new creatures and objects, but all of them uses version 2.6 of gambryo engine, that is afaik cannot be wathed by nifscope or max plugin.
Also in some of the old meshes changed animation files, and even some old model files have changed too, again new ones uses 2.6 version and cannot be imported.
So till the niftools doesn't support this engine version it's better to find old Atlantica client version if you want to use models.
