Post by MaKiPL » Tue Aug 29, 2017 12:17 pm

Version 129KB:

Code: Select all

RGBA 8888
SHIFT: 52, so 0x33 header not 0x34

RGB 24 + 8 alpha
Width: 128 Height: auto

Flip RGB to BGR


Version 65KB:

Code: Select all

RGBA 4444 
SHIFT: 52 

RGB 12 + 4 alpha
Width: 128 Height: auto

Flip RGB to BGR
