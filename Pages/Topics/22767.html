I made tool, try it. Run it in cmd, pass path to bnk and paths to wems or directories with wems. It looks in bnk for such wems and replaces them if found. Make sure wem names are their original IDs. I tested it a bit in DOOM 2016 and it works. Still, it may fail if bnk structure somehow changed with different wwise versions or your wem is differs from game's (I tried just swapping some game sounds), so make sure to backup your bnk.
