Post by deepshit » Thu May 05, 2016 8:20 pm

Code: Select all

comtype bpe

get size long
get ZSIZE long

set NAME string "dump"
string NAME += ".dmp"
clog NAME 8 ZSIZE size
I think this should decompress it.
save this code on a txt file.
open quickbms.
select the txt file.
select the compressed data.
select output folder.
done
