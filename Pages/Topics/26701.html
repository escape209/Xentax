Post by Bigchillghost » Fri Apr 28, 2023 6:04 am

Unpack files with the following BMS cmds:

Code: Select all

endian big
get fileCnt short
for i = 0 < fileCnt
	get Len short
	getdstring Name Len
	get Offset long
	get Size long
	set Ext string Name
	string Ext ! "."
	if Ext == "geom"
		set ZSize long Size
		math Size * 20
		clog Name Offset ZSize Size
	else
		log Name Offset Size
	endif
next i
The rest should be easy.
