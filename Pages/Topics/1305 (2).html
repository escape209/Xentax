Hi mate,

Game Extractor did open the archive for you, as is indicated by the files listed on the left. However, there are a lot of problems with this game because the archives are not the same as PC games - so when opening the archive it has only shown some of the files in the archive.

One solution, if you only want the image files, is to purchase the Full Version of Game Extractor which has a scanner to find TIM2 images, but it isn't in the Basic Version.

I am sorry but there is not much that can be done about this game, and the same goes for many other PS2 and PS1 games
