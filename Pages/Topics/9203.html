Post by RandomTBush » Wed Jul 04, 2012 6:06 am

The Dolphin emulator can view the file systems for Wii/GameCube games. Just set up a folder for the game(s), set that path up in Dolphin, and right-click the game you want to export files from and click "Properties". The "Filesystem" tab has the contents of the ISO, right-click a file and choose "Extract File", or "Extract All Files" if you want to unpack everything.

Post by snakemeat » Tue Aug 28, 2012 8:53 pm

VGMToolbox has an ISO extractor included. It can unpack GC/WII (with ckey.bin) as well.

It has plenty of formats included: "GameCube, Green Book (CD-i), ISO 9660 (PSX/PS2), NullDC GDI (Dreamcast), Opera FS (3DO), Wii Optical Disc, XDVDFS (XBOX/XBOX360)" at the time of this post.

Just open Misc. Tools > Extraction Tools > Generic > ISO Extractor
