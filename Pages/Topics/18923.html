God Eater Resurrection PC/Vita Tool

  • Report
  • Quote
  • Thanks

Post by akderebur » Fri Oct 12, 2018 3:52 pm

Tool for extracting models from God Eater Resurrection, both PC and Vita versions. It might work with God Eater 2 if the format is the same, but I don't know. I only tested it with bunch of character models. Finding the model files is hard since the archive unpacking isn't that great.

Unpacking

Use this script to unpack .rdp or .qpck archive : https://aluigi.altervista.org/bms/god_eater_2.bms

This is the most problematic part. The archive gets unpacked for the most part, but folder names and file extensions are horrible. Most of the stuff has .ixo extension (including models) and you can't identify the model files easily because of that. So just try to load them in the tool until you find a model. From my experience files that have "ND" in their name are mostly models, like : "NPC_003ND.ixo".

3D Models

Load .ixo files with the tool, and the model will appear in the preview, if the file contains model data. You can export the model to SMD format. Noesis works fine with SMD. So I would recommend it for loading/converting.

Overview :



Tool Download : https://www.mediafire.com/file/bi5atmw4 ... r.rar/file

Textures

Textures are in GXT format for Vita and DDS format for PC. Both can be loaded with Noesis. Some texture files don't get unpacked properly though. Btw PC version has higher res textures.
