Here are the 2 bms scripts needed to extract the files from this game.
Run this one first

Code: Select all

open FDDE DAT 0
open FDDE IDX 1

get BPK long 1
get asize long 1
get version long 1
get files long 1
goto 0x40 1
for i = 0 < files
get offset long 1
get null1 long 1
get size long 1
get null2 long 1
get null3 long 1
get null4 long 1
get null5 long 1
get null6 long 1
set name i
string name + .unk
log name offset size
next i
Then run this on the resulting files

Code: Select all

idstring "\x00\x6B\x61\x70"
endian big
goto 0x80
get baseoff long
comtype deflate
math baseoff + 0x88
get files long
for i = 0 < files
set offset baseoff
math offset + 0x80
get size long
