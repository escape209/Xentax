Bullet Girls Phantasia (PC) Tool

  • Report
  • Quote
  • Thanks

Post by akderebur » Mon Mar 23, 2020 12:10 am

I wasn't really interested in this game, but the format was too similar to Star Ocean. So I decided I might as well do a tool with small edits. It supports geometry/skeleton/skinning/animation. Tested with costumes and head models mainly, not sure about other stuff. Usage is similar to SOA tool.

3D Models

Tool overview



How to use
- Select the model file (.asf) you want to load.

- Select the animation file (.aaf) you want to load.
- Animate Position : Will be unchecked by default. It is easier to preview the animations this way. You can check this before exporting if you need to have position keys in the animation.

- Export NEXS : Exports the model in a custom format to be loaded in Noesis. Supports animations. Noesis import script : Nexs Script
- Export IQE : Exports the model in IQE format. How to open/convert IQE: http://forum.xentax.com/viewtopic.php?p=138153#p138153 New script with secondary uv support : http://forum.xentax.com/viewtopic.php?p=143565#p143565

Download : https://www.mediafire.com/file/cj58gk8b ... r.rar/file

An animation preview



Textures

Acewell's noesis script for .aif files was somewhat working. I just made slight changes and added new texture types.
