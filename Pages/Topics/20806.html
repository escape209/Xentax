Noesis Script for 300 Heroes《300英雄》
prev.png (716.41 KiB) Viewed 951 times
Website: http://300.jumpw.com/
Download: http://300.jumpw.com/download.html

Code: Select all

#quickbms Script for .jmp files
#by chrrox
goto 0x32
get FILES long
for i = 0 < FILES
getdstring NAME 0x104
get OFFSET long
get ZSIZE long
get SIZE long
getdstring NULL 0x20
clog NAME OFFSET ZSIZE SIZE
next i
