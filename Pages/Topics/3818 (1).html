offset 0x0C (usually 0x800) is the offset to some data, the count of which is at offset 0x18

Code: Select all

struct
{
    uint32 nameLength;
    char name[nameLength];
    uint32 unknown[3];
};
offset 0x1C is the size in bytes of the NAME_OFF data
the INFO_OFF value is at offset 0x3C
