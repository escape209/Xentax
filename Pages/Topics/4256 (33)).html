Swatchbin Texture Modding Assistant UPDATE

Swatchbin Texture Converter extends with new extension - Swatchbin Texture Modding Assistant interface.
The interface allows to open, browse and replace swatchbin textures, also supports ZIP archive streaming for easy modification of game archives without extracting, thanks to our THFileSystem API (used in MTC).

Swatchbin Texture Modding Assistant features yet:
• Generate template for editing
• Replace swatchbin
• Create backup
• Restore from backup
• Tons of handling conditions
