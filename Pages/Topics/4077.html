Little BMS Script, if it doesnt work post some more fils.

Code: Select all

Get Offset LONG
Get FileSize ASIZE
Get FileName FILENAME

Set Size FileSize
Math Size -= Offset
String FileName -= 5
String FileName += ".DDS"

Log FileName Offset Size
