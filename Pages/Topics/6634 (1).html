Post by chrrox » Fri May 20, 2011 2:40 am

here is the bms script.
you need to use the 64 bit quickbms to extract it all.

Code: Select all

idstring DZIP
comtype COMP_LZF
get unk01 long
get files long
get unk02 long
get tableoff longlong
for i = 0 < files
goto tableoff
get nsize short
getdstring name nsize
get unk03 long
get unk04 long
get size longlong
get offset longlong
get zsize longlong
savepos tableoff
goto offset
get offadd long
math offset + offadd
math zsize - offadd
clog name offset zsize size
next i
