If you're going to use 3D Object Converter to convert models, then you need to buy a register otherwise models are going to have holes in them. You can also try to write a program like 3D Object Converter yourself, but if you want so then you need to do research regarding it yourself.
There are many tutorials on Youtube on how to apply textures to models in Blender. Just follow the tutorial regarding crunch and textures (which is in this topic) and then you can use those textures in Blender.

It can be downloaded here: https://drive.google.com/drive/folders/ ... sp=sharing

To run it, you first need to install the .NET 6 Desktop Runtime. from https://dotnet.microsoft.com/en-us/down ... .0/runtime
