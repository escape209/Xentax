I did it for my friend weks ago. Hope this help!

Code: Select all

# Game: Lost Heroes [PSP]
# by Fatduck     Sept 2012
# http://aluigi.org/quickbms.htm

idstring "2NBF"
get NUMRES long
get OFSBASE long
get BLKSIZE long

for i = 0 < NUMRES
   getdstring RESNAME 0x80
   get OFSRES long
   get NULL long
   get SIZERES long
   math OFSRES += OFSBASE
   log RESNAME OFSRES SIZERES
next i
