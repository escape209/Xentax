Post by Falo » Sun Oct 17, 2010 6:24 pm

ultimaespio wrote:Thanks. I'll upload some sample files and see what people can do with them.

EDIT: How do you use this for KH1?
just double click the exe to see options or see readme,

Code: Select all

kh1_file_dumper.exe kh1.iso KINGDOM.IDX -dT
-dT = dumps all files from "nametbl.txt", but only if it exists.
-dN = dump models from *.ard nametables

these options here are bruteforce names:
-dM = dump music files
-dE = dump event files
-dO = dump other files
