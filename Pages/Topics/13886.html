Post by brendan19 » Sun Jan 31, 2016 4:19 pm

For anyone looking into this

Use this BMS script

Code: Select all

# From Dust (BF format)
#
# Written by Ekey (h4x0r)
# http://www.progamercity.net
#
# script for QuickBMS http://quickbms.aluigi.org

idstring "ABE"
goto 0x18
get TABLEOFFSET long
goto 0x2C
get FILES long
goto TABLEOFFSET

for i = 0 < FILES
   get DUMMY long
   get DUMMY longlong
   getdstring NAME 108
   get OFFSET long
   get DUMMY long
   get SIZE long
   getdstring TRASH 68
    log NAME OFFSET SIZE
next i
