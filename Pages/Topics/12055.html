Hi guys! Here's maxscript to import PC BlackGold Online models with bones+weights (3ds max 2009-2012)

Script to extract files by Ekey
viewtopic.php?f=10&t=11471&hilit=BlackGold

1. Run Black_Gold.ms
2. Select t_pose.xskt file (bones, it's importand to select t_pose file, since others are animation)
3. Select model file .xmod, you want to import
4. Select .mtl file of model you imported (have same name as model)

Done!

P.S. Report bugs
