Here's a quickbms script
G1M extracter

Code: Select all

endian big
for i
findloc start string "G1M_0034"
goto start
print "%start%"
savepos offset
getdstring null 0x8
get size long
set name i
string name + ".g1m"
log name offset size

next i
G1T extracter

Code: Select all

endian big
for i
findloc start string "G1TG0060"
goto start
print "%start%"
savepos offset
getdstring null 0x8
get size long
set name i
string name + ".g1t"
log name offset size

next i
I have altered chrrox's Dynasty Warriors 7 quickbms script.
