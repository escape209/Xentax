Post by merlinsvk » Sun Apr 29, 2012 5:22 pm

Ehm, guys, command prompt doesn't like spaces in the path. That's the problem.
So remove them or use quotes, like this:

Code: Select all

ttarchext -x -b -V 8 52 "C:\Some Path With Spaces\blah.ttarch" "C:\The Walking Dead Tools\TTG Tools\Input"
