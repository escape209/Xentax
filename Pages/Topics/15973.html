AceWell wrote:for AedMale_lowpoly.(Geometry).bin use offzip
instead of using offzip to decompress use this bms script so you can run on entire folders and keep the file names

Code: Select all

# script for QuickBMS http://aluigi.altervista.org/quickbms.htm

comtype zlib_noerror
get ZSIZE asize
get NAME basename
get EXT extension
string NAME + "_decomp."
string NAME + EXT
clog NAME 0 ZSIZE ZSIZE
freakshow wrote:can u create a noesis script?
