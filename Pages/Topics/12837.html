Unity Engine Patch And Unpacker 2.2
By. MiRiKan

This will unpack .assets file for Modding.
Support Almost of Unity .assets FILES! (Unity 4 to Unity 5!)

This program need .net framework 4.5

How To Use
Just Drag and Drop, Easy to use.
Just Drag to this EXE file what .asssets file you wanna unpack or repack.
and type 1 for unpack. (program will create unpacked files, .txt file with data - do not delete it)
then check created folder, you can find many of files - unpacked, and patch folder.

EX: 8-TYPE_21-Monsters, in this file. 8 is a count of all unpacked files. TYPE_21 says type is 21. And Monsters is Name of file.

So if you wanna modding this file, edit it and replace it patch folder (also I recommend do not delete original unpacked file)
LIKE THIS:

Name_of_File.Type.

Program will read when repacking this.

Unpacked files are original data - not decoded. But you can change or Modding it using the unity engine and Unity Studio beta from Chipicao
It does not decode any images or texts because this is just for repacking and Modding.

sry my bad english, but it's very useful and will help you guys to modding games made by unity!

OK. this is another tool for edit TEXT from Unity .assets files, which is 49 type.

After all of works, just drag txt file onto UnityType49_build.exe
this will create brand-new .49 type file.
*YOU HAVE TO MATCH TXT FILE'S NAME WITH ORIGINAL 49 TYPE FILE.
