Its of the PSP console,right?
If yes,then-

1.Download JN 2.1.4g : blog/?p=62
2.Click File>Load
3.Browse to the ISO image and select it directly
4.Tick the 'Ultrafast Stripped Scanner'
5.U'll get sum *.wav files in the ripped folder
6.Get WRar : www.rarlab.com/download.htm‎
7.Install it
8.Get the AT3 Codec : http://www.free-codecs.com/download/son ... _codec.htm
9.Right Click on the codec installing exe and extract it using WinRar
10.Browse thru the extracted files and locate the *.inf file
11.Right click the INF file and install it
12.Get GWave : http://www.goldwave.com/release.php
13.Install it and start batch processing ur *.wav(AT3) files into a more convenient format like MP3
14.Done!!!
