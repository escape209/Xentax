I'm assuming that the animations (and rigging?) are in the .motion files but cannot confirm.
My end goal is to be able to play through the animations of a character and then export an OBJ or STL of the model in that pose so that I can 3D print it (see, for example: https://www.youtube.com/watch?v=JTGOiiK4qSA)

I've tried using Milkshape 3D as in that video linked but as the files are not from UE the .motion files are not recognized.
