Tool for decrypt / encrypt assets\Art.dat

Code: Select all

[Usage]
        PPTool <szMode> <szVersion> <szInFile> <szOutFile>

Code: Select all

[Modes]
        -d - Decrypt
        -e - Encrypt

Code: Select all

[Versions]
        -1 - 1.0.3x
        -2 - 1.0.4x

Code: Select all

[Example]
        PPTool -d -1 Art.dat Art.dec
