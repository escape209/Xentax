ok there's another way but i still didn't test it if anyone wanna try it

first download Frostbite editor https://frostytoolsuite.com/

if this version doesn't work download the 1.0.5.1 version from github .

after downloading the editor . open it and click *new* search for your game Path (Original Game fully updated not cracked and add it )

click *select*

the use the following serial if you are asked for encryption key : https://drive.google.com/file/d/1pVjnuJ ... sp=sharing

it should log in to the game now and give you access to the tree structure of the game . look up the folder sound and check the sounds you want and export to *wav*

a guy did extract 2 audios using this method not that i tired myself since i don't have the original game :/
