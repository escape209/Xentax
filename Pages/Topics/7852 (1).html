Post by chrrox » Sun Jan 08, 2012 1:02 am

here is the bnd extractor script

Code: Select all

endian big
idstring "BND307D7R6"
goto 0x10
get files long
get start long
goto 0x20
savepos tbl
for i = 0 < files
goto tbl
get unk01 long
get zsize long
get offset long
get unk02 long
get nameoff long
get size long
savepos tbl
math nameoff + 3
goto nameoff
get name string
if zsize == size
log name offset zsize
else
clog name offset zsize size
endif
next i
