you can decompress the bin with this bms script:

Code: Select all

# script for QuickBMS http://aluigi.altervista.org/quickbms.htm

get NAME basename
string NAME + _decmp.bin
getdstring MAGIC 8
get ZSIZE long
get SIZE long
clog NAME 0x10 ZSIZE SIZE
