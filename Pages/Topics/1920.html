Post by PXR » Sat Aug 19, 2006 12:47 pm

I didn't see this thread, but with a quick look on the supplied archive; this is what I find out:

Code: Select all

04 - version? (1)
04 - number of files 
24 - null

// for each file

	06 - filename
	26 - unknown
	04 - file offset
	04 - file length
	04 - unknown

// for each file

	 x - file data
	 x - null padding
BMS-script:

Code: Select all

ImpType Standard ;
Get TEMP Long 0 ;
Get FILENUM Long 0 ;
GoTo 32 0 ;
For F = 1 To FILENUM ;
GetDString FNAME 6 0 ;
SavePos TEMP 0 ;
Math TEMP += 26 ;
GoTo TEMP 0 ;
SavePos FOFFSETX 0 ;
Get FOFFSET Long 0 ;
SavePos FSIZEX 0 ;
Get FSIZE Long 0 ;
Get TEMP Long 0 ;
Log FNAME FOFFSET FSIZE FOFFSETX FSIZEX ;
Next F ;
