Ok, that was an easy one, I think :

Code: Select all

ImpType Standard ;
Get FN Int 0 ;
For T = 1 To FN ;
SavePos FSO 0 ;
Get FS Long 0 ;
SavePos FOO 0 ;
Get FO Long 0 ;
GetDString FName 16 0 ;
Log FName FO FS FOO FSO ;
Next T ;
Try this file (Use Custom BMS on..) on a LIB file.
