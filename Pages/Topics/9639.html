+ compressed.. Here useless script.. + need more examples.

Code: Select all

# Dreamy Theater Extend (PS3) (FARC format) 0.1a (Unfinished)
#
# Written by Ekey (h4x0r)
# http://forum.xentax.com
#
# script for QuickBMS http://quickbms.aluigi.org

endian big

idstring "FARC"
get TABLESIZE long
set FILES = TABLESIZE
math FILES -= 20
math FILES /= 30
goto 0x1C

for i = 0 < FILES
   get NAME string
   get OFFSET long
   get ZSIZE long
   get SIZE long
   log NAME OFFSET ZSIZE
next i
PS : for each files same header (0x10 bytes - maybe key for decrypt??) - 6D4A249C8529DE62C8E3893931C9E0BC
