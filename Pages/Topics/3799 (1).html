Post by Karpati » Wed Nov 11, 2009 12:37 pm

I think the best export format is the Wavefront's .obj/mtl format. The game modders (and others) using this format to import the (fully textured) objects into other 3d modeling programs like 3D Studio Max, Cinema 4D, Maya, etc.

I can't support the .max format, because it it totally unpublished.
