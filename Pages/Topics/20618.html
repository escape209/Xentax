Post by Acewell » Thu Jun 06, 2019 12:24 pm

here is bms script.

Code: Select all

# script for QuickBMS http://aluigi.altervista.org/quickbms.htm

get FOLDER basename
get UNK long
get FILES long
goto 0x18 0 seek_cur
for i = 0 < FILES
    get NAME string
    string NAME p "%s\%s.msv" FOLDER NAME
    goto 0x19 0 seek_cur
    get OFFSET long
    get SIZE long
    get UNK2 long
    log NAME OFFSET SIZE 
next i

Post by Pingu » Thu Jun 06, 2019 8:26 pm

Also a little side-note, if you replace this line in Acewell's script from:

Code: Select all

string NAME p "%s\%s" FOLDER NAME
to

Code: Select all

string NAME p "%s\%s.msv" FOLDER NAME
That will extension the files to .msv which is playable with foobar + vgmstream component.
