Szkaradek123 wrote:Hi
Here is unpacker and importer for models from this game.
It requires Blender version 249 and Python 2.6.
How use:
1.unpack importer.
2.use CriToolpack.exe on rom0.cpk (tool made by viewtopic.php?p=87026#p87026 )
3.in Blender Text Window press alt+p and select:
- one unpacked file like "FILE_00658" and wait until this all type files will be too unpacked.
- go to folder "char/unpack" and select jmdl for import textured and rigged meshes

Small update: fix rotMatrix for bones (convert radians to degrees)
