That program does not appear to extract the files correctly.
It is just extracting war files not decompressing them.
this is a bms script that does the same thing only much faster.

Code: Select all

endian big
getdstring name1 0x40
get null long
get tableoff long
get null2 long
get files long
goto tableoff
#comtype lzss
for i = 0 < files
getdstring name 0x40
get unk01 long
get unk02 long
get unk03 long
get offset long
get unk05 long
get size long
get zsize long
get unk08 long
log name offset zsize
next i
