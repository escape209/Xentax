CFF file it's an archive of files zlib compressed.

Code: Select all

4 - Header (0x12DD72DD)
4 - Archive Version? (3)
12 - null

// for each files
   4 - File ID?
   2 - (1)
   4 - Compressed size
   2 - (1)
   4 - Uncompressed size
   x - Data File (zlib compressed)

Post by baccello » Sat May 05, 2007 1:17 am

Use this BMS script.

Code: Select all

ComType ZLib1 ;
Set START Long 20 ;
GoTo START 0 ;
Do ;
Get ID Long 0 ;
Get FOO Int 0 ;
Get CS Long 0 ;
Get FOO Int 0 ;
Get US Long 0 ;
SavePos OFF 0 ;
If CS <> 0 ;
CLog "" OFF CS 0 0 US 0;
Math OFF += CS ;
GoTo OFF 0;
EndIf ;
While CS <> 0 ;
