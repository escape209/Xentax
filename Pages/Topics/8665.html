This is my try to write an QuickBMS script to export/import BMPs from .000 files.

Code: Select all

# Dark Seed II .000 file
# script for QuickBMS http://quickbms.aluigi.org

get FILES short
for i = 0 < FILES
getdstring NAME 0xC
get SIZE long
get OFFSET long
log NAME OFFSET SIZE
next i
