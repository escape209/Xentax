Modelbin Collector 1.0.4 - FM7 Support BETA

Hello there!
After release of FM7 we have been working on Modelbin Collector update since the old one will not work in most cases.

1.0.4 FEATURES
• Added partial support for Forza Motorsport 7 UNENCRYPTED/DUMPED car archives. (see guide below)
• Reworked library lookup mechanism
• Library Manager - allows you to manage your library archives.



Quick guide for FH3:
Well, nothing actually changed when it comes to library setup.
1.) Open MTC
2.) File -> Library Setup...
3.) Add archives from (...\media\cars\_library) [except _pri_44]
4.) Close dialog
5.) Collect FH3/FM6 archives

Quick guide for FM7:
1.) Download unencrypted and repacked Forza 7 Library if you don't have your own. (Links -> Download -> Forza 7 Library Collection)
2.) Extract
3.) Add archives from downloaded library to Library Manager (File -> Library Setup...)
4.) Unpack FM7 car archive using quickBMS
5.) Pack back again to ZIP
6.) Collect FM7 Archives

WARNING Do not load both libraries at once (FH3/FM7), it may heavily slow down collection process.

BTW, does anyone found a way to access original ZIP archives? All we got is this:

Code: Select all

invalid CEN header (unsupported compression method: 21)
