Noesis shows error because it read the animation but can't find the corresponding bone in pab.
I have checked that it loaded two bones correctly,"Bip01" and "Bip_Weapon_R".

The paa file is under folder /character/motion/1_pc/7_pvw/weapon/45_spear/. And only 24 bones have animation. So I think you should find the skeleton of this weapon "45_spear" to load animation properly.

Or, if you find any file which contain the following hex, it should be the spear's skeleton.
"C9 80 CF EC"
