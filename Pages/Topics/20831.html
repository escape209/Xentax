Here. Quickbms script for decompressing them.

Code: Select all

get BNAME basename
get TotLen long
putvarchr MEMORY_FILE TotLen 0
log MEMORY_FILE 0 0
append
Do
get ChunkLen long
SavePos OFFSET
clog MEMORY_FILE OFFSET ChunkLen 10000000
math OFFSET + CHUNKLEN
goto OFFSET
get ChunkLen long
goto -4 0 SEEK_CUR
While ChunkLen > 0
append
get MEM_SIZE asize MEMORY_FILE
log BNAME 0 MEM_SIZE MEMORY_FILE
