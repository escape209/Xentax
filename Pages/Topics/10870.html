Post by luxox18 » Fri Oct 18, 2013 9:06 pm

simple explanation:

There are different methods for obtain models from console games.
For example in xbox original and xbox360 you can obtain meshes and textures using "PIX" for the xdk console
in ps3 you can obtain meshes and textures using GPAD tool (SCE tool), but for this you need a debug console + decrypted and resign eboots ( very difficult because the console crashes when you try to enter in the RSX vram)
how to watch or dump the ram data from ps3....
for convert a console from CEX to DEX there are many easy methods. I recommend the REBUG tool for this (this not take more than 5 minutes)
you need decrypted and resigned eboots (debug) + SCE tools (Target manager, debugger)

Then run your console in debugger mode with target manager, load the game from your pc to the console with debugger tool (proDG Debugger for ps3), with this you can watch the entire ps3 ram.

copy all hex values (or only the 3d model values) from the process window to the pc using a hex tool. (maybe you can export all data directly , but for me don't work)

NOTE: not all games works in Debugger mode (uncharted , the last of us,etc.)
