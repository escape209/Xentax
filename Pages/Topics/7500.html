GTA IV Console Texure Editor is an advanced texture dictionary editor, with support for console GTA IV and Episodes From Liberty City games.



Current public version: 1.0 public
Designed and made by: Dageron

Supported platforms: Xbox360 and PS3
Supported formats: *.xtd, *.xhm, *.ctd, *.chm

Technical issues:
* Program is still in testing development stage. Possible buggs will be corrected in future versions.
* Replaced the texture should have same parameters as the original one.
* PC-version is not supported.
