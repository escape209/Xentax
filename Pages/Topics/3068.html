Post by aluigi » Tue Oct 20, 2009 5:44 pm

if someone is interested to the extraction of the ALLTXT.PAK file use the following script:

Code: Select all

get DUMMY long
get FILES long
get PAK_SIZE long
get DUMMY long
get DUMMY long
get DUMMY long
savepos INFO_OFF

for i = 0 < FILES
    goto INFO_OFF
    get NAMEOFF long
    get OFFSET long
    get SIZE long
    get DUMMY long
    get DUMMY long
    get DUMMY long
    get DUMMY long
    savepos INFO_OFF

    goto NAMEOFF
    get NAME string

    log NAME OFFSET SIZE
next i
