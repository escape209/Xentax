Info about the game: Windows Store game: https://www.gangstarneworleans.com/



After you downloaded it, follow the next site to pinpoint the downloaded files: https://www.intowindows.com/how-to-acce ... indows-10/

You'll find the packages in a folder similar to this one: C:\Program Files\WindowsApps\A278AB0D.GangstarNewOrleans_1.2.6.4_x86__h6adky7gbf63m\data\packages

They are compressed vArc files that can be extracted via the bms script called VisionPackage_vArc.bms.

Download the bms script: https://www.dropbox.com/s/t70b2tel8pa66 ... c.zip?dl=0

It is still work in progress, but the mesh imports, normal and uvw's to

Still working on the submeshes, bones, weights, etc..
