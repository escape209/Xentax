atsushukun wrote: ↑Fri Sep 16, 2022 5:36 pm...keeps saying virus detected even tho i turned off al of the security settings is there any fix to that?...
It is a false positive, the executable is packed in a manner to protect its source code from de-compile/disassemble it, you will have to find a way to download it, besides removing the settings for real-time protection and allow the download the browser to download it, there shouldn't be any other issues to access it, you'll have to put it in to a folder and add that to a allowed location in your AV software so you can execute it freely, so it wont get deleted again.
