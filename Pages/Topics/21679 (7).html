Post by Yretenai » Tue Apr 14, 2020 1:03 am

CosmicDreams wrote: ↑Mon Apr 13, 2020 10:30 pm Does this tool support the .ZL_ files from marvel ultimate alliance 3?
MUA3 ZL files are stream compressed files, use Cethleann.Gz.

Code: Select all

Cethleann.Gz.exe --stream --prefixed-size -t Deflate {ZL files}
i.e.

Code: Select all

Cethleann.Gz.exe --stream --prefixed-size -t Deflate 0400_DLC_00_mdl.bin.ZL
