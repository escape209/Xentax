Download the latest VIS3 module that includes "unpakke.ini"
Open "unpakke.ini" and check the "magic" value. There is a list of magic codes for all the supported games in the file. Take the "A new beginning" hash and replace the magic value with it.
Now try to unpack the files again and they should have no problems (I've already tried ).
