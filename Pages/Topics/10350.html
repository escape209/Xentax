Well here my simple packer.



And small test with edited language file



1.) Run VFS Unpacker
2.) Unpack files
3.) Edit files (Dont rename __Unknown folder and filenames inside)
4.) Run VFS Packer
5.) Select directory with unpacked files
6.) Select output VFS file
7.) Press Build
8.) Profit

Enjoy
