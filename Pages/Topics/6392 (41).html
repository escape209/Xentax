Ok gonna make an extractor for doa5u now. Thanks for that post it explained it all.
to hold people over for now here is a ram extractor.

Code: Select all

goto 0x23D61A0
endian big
set tmp 0
do
get tmp long
get offset long
math offset - 0xA0000000
get size long
log "" offset size
while tmp != 65535
