Dorigon wrote:I can't use winrar or 7zip... I get allways error. I'm triying with offzip but nothing
run this from the command line:

Code: Select all

offzip.exe -a -z 15 ctrTango_long.zlib . 0
you could also use a bms script (see quickbms)

Code: Select all

get unsize long
get zsize asize
math zsize -= 4
clog "uncompressed.dat 4 zsize unsize
