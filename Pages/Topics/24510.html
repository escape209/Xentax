I have finished my The Legend of the Condor Heroes .col and .mesh loader modules and I have released the following programs as web updates:

- 3D Object Converter v8.013 (Windows)
- i3DConverter v4.103 (macOS)
- i3DConverter v2.103 (Linux)

How to get the 3D Object Converter v8.013:
Download the 3D Object Converter from http://3doc.i3dconverter.com and install it or download and use the portable version.
After it just use the Help/Check for updates... function to get the v8.013.

How to get the i3DConverter macOS v4.103:
Download the i3DConverter from http://www.i3dconverter.com and install it.
After it just use the Help/Check for updates... function to get the v4.103.

How to get the i3DConverter Linux v2.103:
Download the i3DConverter from http://www.i3dconverter.com and install it.
After it just use the Help/Check for updates... function to get the v2.103.
