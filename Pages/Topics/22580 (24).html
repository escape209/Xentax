Jan666 wrote: ↑Thu Aug 03, 2023 12:08 pm Can someone tell my how i can extract the Textures of tlou part 1 pc version, on the alphazomega tool, the extractet textures have black stripes and only 17 kb theire are useless
The black striped textures are the ones merged in the models file (low res ones) in order to preview and export the high res ones, you have to extract the files and create the original folder structure like this for example (world Bill) otherwise Noesi can't find correct path and shows you lowres ones:

also you have to edit the fmt_nd_pak.py Noesis script and set the correct path where you have extracted the files for example my one is:
"TLOUP1": "D:\\ExtractedGameFiles\\tlou1\\build\\pc\\main\\",
or you can use tloupc textures browser tool
