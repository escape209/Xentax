If you open the new .mdx2 file, my program extracts the .png file(s) from the .mdx2 files automatically if the Tools/Options/Import #2/Tianji Online – Extract the .png file(s) from the .mdx2 files after load parameter is on (default=on).

- 3D Object Converter v10.010 (Windows)

How to get the 3D Object Converter v10.010:
Download the 3D Object Converter from http://3doc.i3dconverter.com and install it or download and use the portable version (if you don’t have it yet).
