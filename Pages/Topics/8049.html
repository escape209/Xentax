Post by destrator » Sat Jul 07, 2012 12:45 pm

Download and install Microsoft Visual C++ 2010 Redistributable Package. This is necessary to run the unpacker.

Then download rqaf_extract.zip, extract it and place rqaf_extract.exe to root of Royal Quest game.

Files of game will extracted to data.extracted folder in root of the game.

If you find errors, please contact me.

Post by destrator » Sun Jul 08, 2012 4:52 pm

I checked it out.

Total size of files in .toc table is 1,856,860,055.
Total size of valid files is 1,784,384,645.
Total size of invalid files (with index 0xFFFF or 0xFFFE) is 72,475,410.

Total size of files in .af files on hard drive is 1,832,910,848. Subtract from this value service data of archives and you'll get the value 1,784,384,645.

Files with index 0xFFFF or 0xFFFE are obsolete or are somewhere else. They are not in the .af packages.
