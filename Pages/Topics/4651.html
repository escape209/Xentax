Post by chrrox » Wed Jun 23, 2010 10:55 pm

ah it must be zlib found that 58 85 can be a zlib header.

here is a bms script if there is some file with names in it let me know or post the .dol file here as it might contain the names.

Code: Select all

#quicbms script
endian big
for i = 0 < 0xFFFF
findloc start string "ZLB"
goto start
get zlb long
get unk01 long
get size long
get zsize long
savepos offset
set name i
string name + .ext
      clog name OFFSET ZSIZE SIZE
math offset += zsize
goto offset
next i
