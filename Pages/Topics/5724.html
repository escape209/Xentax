Hi.

I'm working on Birth By Sleep file formats in these days: DAT (the main archives), ARC (some sub-archives that contain textures and 3D models or some data) and CTD (that store some texts and subtitles).

The ARC format it's easy and I've already created an packer/unpacker.
The CTD format it's a bit hard to understand, but I've some ideas to know how it work.
The DAT files are the most important archives of the entire game! They contain all the data of the game and the file format it's a pain but I'm working on it!

I don't know if I've the free permissions to post on "Tools & Programs" or to post on blog on my first post and I've decided to write here.
If anyone is interested I can share some informations about these files
