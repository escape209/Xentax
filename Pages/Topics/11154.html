Unity is a game development ecosystem: a powerful rendering engine fully integrated with a complete set of intuitive tools and rapid workflows to create interactive 3D and 2D content; easy multiplatform publishing; thousands of quality, ready-made assets in the Asset Store and a knowledge-sharing community.
You can read more about it on http://www.unity3d.com.

As this game engine grows more and more popular, I think we should make a list of Unity games that have interesting models, which can be imported with my MAXScript - info here and direct download here

Instructions on downloading and extracting games
  • PC, OSX & Linux - download the game from wherever and the contents should already be available in a Data folder
  • Android - Since Chrome doesn't allow unverified extensions anymore, the APK Downloader won't work unless you're using Canary.
    Luckily there's an easier way with http://apps.evozi.com/apk-downloader/. Simply enter the app URL from Google Play and start the download.
    Change the extension from .apk to .zip and extract.
    Note: some games/apps download additional content after they are installed. Usually it's in the form of .obb files which are just zip archives.
  • iOS - the only way I know is through iTunes; install it, create an account (preferably US or UK) and browse the store. Downloaded apps are stored in \My Music\iTunes\iTunes Media\Downloads
    Change the extension from .ipa to .zip and extract them.
    Note: some games/apps download additional content after they are installed. I don;t know of any way to get this content without actually installing them.
  • Xbox 360 - use the usual Xbox tools to extract the game (wxPirs). Files are inside a folder called Media instead of Data
  • Web - this is probably the hardest one, or at least the most tedious
    • Download, install and run URLsnooper2 - you may have to restart
    • Under "Protocol filter" select "Show all" and then press "Sniff Network"
    • Load the game in your browser - if it doesn't work in Chrome try IE or FF
    • Some Web games won't load all assets at once. For example, a car configurator will download additional files after you change car options
    • Select the files you need in URL Snooper, right click and copy to clipboard. Then you can download them one by one or use a download manager
    • Put all files in one folder and use aluigi's QuickBMS and unity3d extraction script. If it doesn't work try the Unity 3D Obfuscator
. Also have a look at my modified version of the script that fixes some issues: viewtopic.php?p=93230#p93230
