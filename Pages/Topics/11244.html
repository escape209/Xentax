the ATTRIBUTEZ.TXT file is encrypted by blowfish with the key "1212"

use this script to decrypt:

Code: Select all

encryption blowfish "1212"
get NAME filename
get SIZE asize
string NAME += ".dec"
log NAME 0 SIZE
