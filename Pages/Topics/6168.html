Post by XpoZed » Wed Mar 02, 2011 5:56 pm

Unpakke is a small and easy to use tool for the modification of resource files. It provides an elegant solution to the problem of distributing large archives, as it gives the ability to distribute only the modified files.

More information can be found here : http://nullsecurity.org/unpakke/

Current version: 1.2 (build 1008)

Yet another module for unpakke (now version 0.4b) can be downloaded from nullsecurity.org
It's called upkk_pak_t2.dll and with him you can repack the "Dracula: Origin"'s PAK archives.
Unfortunately if you try to repack sounds_en.pak the game will not have any sound on the characters talk. The problem is mainly in the sounds_en.pak's logic (but ofcourse ), and not in the unpakke module!

Other than that, you can do whatever you like.
