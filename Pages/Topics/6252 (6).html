I know this thread is old, but does anyone with experience using Sgwstool mind explaining how to use it, please? It's all in Asian characters I don't understand.
The tools are simple
From Left to Right
Open - Open *.bin files from Dw7

-- With single top level selected
Import Data - Raw data block import
Export Data - Raw data block export, extension is *.data, it uncompress the block and export. so you can export *.g1m or *.gtm
-- With sub data selected
Import Resource - If it's picture, you can import picture
Export Resource - If it's picture, you can export picture to bmp or png format. if it's mesh you can export to obj (for mesh, icon can be gray, but you can still export with right mouse button menu in the right panel). but obj export seems not perfect
