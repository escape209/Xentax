Post by daemon1 » Mon Oct 26, 2020 1:23 pm

wuyilurenjia wrote: ↑Mon Oct 26, 2020 11:41 am After I use spiderman allassets.txt, it showed "asset not found:XXXXXXXXXXXX"
And spiderman models.txt, it showed "could not find the file g02s000", but there is only g00s000 ~ g00s018 in my asset_archive floder.
Its bad idea to use allassets.txt, because only models and textures are supported by the tool, and this list only included for reference and research.

If you get "could not find the file g02s000", it means your TOC file has references to it, and this file must be present in your folders.
