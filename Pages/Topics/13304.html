Post by chrrox » Tue Jul 25, 2017 2:38 am

quickbms script
all files you sent only contained 1 file so can't do much more on multiple file archives without examples.

Code: Select all

comtype deflate
idstring BARC
get unk01 short
get unk02 short
get namestart long
get fsize long
get unk03 long
get unk04 long
get fileTBL long
get null long
get unk05 long
get NAME string
set offset namestart
math offset += unk04
goto fileTBL
get size long
get zsize long
get base long
clog NAME offset zsize size
you can also just run offzip on these files
offzip -1 -a -z -15 c:\mfile.arc c:\export_folder 0x0
