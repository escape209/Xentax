Post by Mr.Mouse » Mon Oct 27, 2003 11:43 pm

I think I cracked it, let's say 90%. The .cdds files are indeed some sort of collection of .dds files, files that are readable with DirectX Texture Tool. In a way, for each texture is a MIP map present in the archive. This is just pre-resized texturing for any given texture, starting from original size to a 1 by 1 format, in size-halfing steps. I have written a small app that can handle the .cdds files and extract the files in .dds format, although the original filenames, the starting point of the authors, were .tga files. I will create an entry in the XeNTaX' OpenGRAF database, Open Game Resource Archive Formats, which I intend to fill with all my and others' cracked formats. Captain Corny will work on an implementation of .cdds files in OpenMex.
