Post by friendsofwatto » Tue Oct 05, 2004 3:15 pm

Well based on Simcity 4, which uses the same archive format, there are no filenames stored with the files. However, there are a *few*, a very minor *few*, that do have filenames stored with them.

The version of Game Extractor you can download from http://www.modthesims2.com, or from my website, will display filenames for the files that have them.

In regards to DatGen, download the latest version (0.7) to be able to read the archives. (http://www.datgen.info)
