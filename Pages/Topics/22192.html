Blood & Truth tool.

Supports skeletal & static models. Textures are GNF, so no need to convert them.

Usage: drop .model file onto the tool, or run from command line with 1 parameter: model file name.
If .skeleton file will be present next to model, it will be converted with skeleton.
Rename generic skeleton to convert generic models.
